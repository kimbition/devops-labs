# Day 03 Linux 进程、端口和服务排查

## 今天学到的概念

- PID：Linux 给每个进程分配的编号。
- 后台进程：不占用当前终端继续运行的进程。
- 端口：服务监听网络请求的入口编号。
- 监听端口：某个进程正在等待连接的端口。
- 端口冲突：两个进程想使用同一个端口，后启动的通常会失败。
